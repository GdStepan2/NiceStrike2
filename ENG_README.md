# 🎮 System Requirements

Below are the approximate system requirements for running the game comfortably.

---

## ✅ Minimum Requirements

| Component | Requirement |
|---|---|
| **OS** | Windows 10/11 64-bit |
| **Processor** | Intel Core i5-4570 / AMD Ryzen 3 1200 |
| **Memory** | 8 GB RAM |
| **Graphics** | NVIDIA GTX 1050 Ti 4 GB / AMD RX 560 4 GB |
| **DirectX** | Version 12 |
| **Network** | Broadband Internet connection |
| **Storage** | 6 GB available space |

---

## ⭐ Recommended Requirements

| Component | Requirement |

## 🌟 RTX / DLSS Support

The game supports **RTX Ray Tracing** and **DLSS**.

> An **NVIDIA GeForce RTX** graphics card is required to use **RTX** and **DLSS** features.
